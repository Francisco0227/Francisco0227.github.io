1. Introducción a HTML

¿Qué es HTML?

HTML es como el esqueleto de madera de una casa en construcción. No tiene pintura, ni muebles, ni papel tapiz; solo le dice al constructor dónde van las habitaciones, las puertas y las ventanas. Es la estructura cruda de una página web.
Referencia: MDN Web Docs - "Conceptos básicos de HTML"

Explica las etiquetas "head" y "body".

El head (cabeza) es como el cerebro o la configuración oculta de la página. No lo ves en la pantalla, pero guarda información importante como el título que aparece en la pestaña del navegador y los enlaces a los estilos.
El body (cuerpo) es la página web física. Todo lo que ves, lees y a lo que le das clic está dentro del body.
Referencia: W3Schools - "HTML Head" y "HTML Body"

Etiquetas Básicas

Explica las etiquetas section, article, aside, header, footer

Imagina que estás leyendo un periódico de papel.
El header es el título gigante hasta arriba con el nombre del periódico.
El footer es la letra chiquita hasta abajo con la fecha y el número de página.
Un section es una página o sección completa, como la "Sección de Deportes".
Un article es una noticia específica e independiente dentro de esa página (una historia que tiene sentido por sí sola).
Un aside es un pequeño recuadro a un lado de la noticia principal con un dato curioso extra.
Referencia: MDN Web Docs - "Estructura de documentos y sitios web"

2. Introducción a CSS

¿Qué es CSS?

Si el HTML es el esqueleto de madera de la casa, el CSS es el decorador de interiores. Es la pintura, las alfombras, las lámparas y los muebles.
Referencia: W3Schools - "CSS Introduction"

¿Cuál es el propósito de CSS?

Su propósito es hacer que las cosas se vean de una manera específica. Controla los colores, los tipos de letra, los tamaños y en qué parte de la pantalla se acomodan las cosas. (En el caso de esta página, su propósito es lastimar tus ojos).
Referencia: MDN Web Docs - "¿Qué es CSS?"

¿Cómo aplicas un CSS a un HTML (explica las 3 opciones)?

  1. En línea (Inline): Es como un tatuaje temporal. Pones el estilo directamente sobre una sola palabra o párrafo en el HTML.
  2. Interno (Internal): Es como pintar una sola habitación. Escribes los estilos dentro del `head` del documento HTML, y solo afecta a esa página en específico.
  3. Externo (External): Es como un libro de reglas maestro. Creas un archivo completamente separado (como `estilos.css`) y lo enlazas. ¡Esto puede darle estilo a cientos de páginas al mismo tiempo!
Referencia: MDN Web Docs - "Cómo funciona CSS"

¿Qué es un selector? ¿Qué es una regla CSS?

Un selector es "a quién" le estás hablando (Ejemplo: "¡Oigan, todos los párrafos!").
Una regla es "qué" le estás diciendo que haga (Ejemplo: "¡Pónganse de color morado!").
Referencia: MDN Web Docs - "Sintaxis CSS"

CSS Básico

Explica los selectores:

  • Selector de Etiqueta (Tag): Apunta a todos los elementos de un mismo tipo. (Ejemplo: "Todas las sillas de la casa se vuelven azules").
  • Selector de Clase (Class): Apunta a cualquier cosa que tenga una calcomanía específica pegada. (Ejemplo: "Cualquier persona con una calcomanía 'VIP' recibe bebida gratis").
  • Selector de Etiqueta y Clase: Muy específico. (Ejemplo: "Solo las SILLAS que tengan una calcomanía 'VIP' se vuelven azules").
  • Selector de Id: Apunta a UN SOLO elemento único e irrepetible. (Ejemplo: "La persona con el número de pasaporte 12345, por favor póngase de pie").
Referencia: MDN Web Docs - "Selectores CSS"

¿Qué es una clase? ¿Qué es un id? ¿Cómo se relacionan con CSS?

Una clase (class) es como la playera de un equipo de fútbol; muchos elementos diferentes pueden ponérsela. Un id es como tu CURP o número de seguridad social; solo UN elemento en toda la página puede tenerlo. Se relacionan con CSS porque son las formas principales en las que "apuntas" a partes específicas de tu página para pintarlas de diferentes colores sin afectar al resto.
Referencia: W3Schools - "CSS Selectors (id and class)"

Explica el Modelo de Caja (Box Model) de CSS. Muestra un ejemplo.

Imagina que cada cosa en una página web es un cuadro de fotografía en la pared.
1. Contenido (Content): La fotografía en sí.
2. Relleno (Padding): El cartón o paspartú blanco que hay entre la foto y el marco de madera.
3. Borde (Border): El marco de madera en sí.
4. Margen (Margin): El espacio de pared vacía que dejas alrededor del marco para que no choque con otros cuadros.

¡Este es el contenido! Nota el padding (espacio dentro del borde), el borde grueso punteado, y el margen (el espacio invisible afuera que lo empuja lejos del otro texto).
Referencia: MDN Web Docs - "El modelo de caja"

Explica las media queries

Las media queries son la forma en que un sitio web cambia de forma (como un Transformer). Es una regla que dice: "Oye, si el usuario está viendo esto en la pantalla chiquita de un celular, apila todo en una sola columna. Si está en una televisión gigante, pon todo lado a lado para aprovechar el espacio".
Referencia: MDN Web Docs - "Consultas de medios CSS (Media queries)"

CSS de Diseño (Layout)

Explica display: grid

Grid es como un tablero de ajedrez o una hoja de cálculo de Excel. Te permite acomodar cosas en filas y columnas perfectamente alineadas al mismo tiempo (de arriba a abajo y de izquierda a derecha). Es ideal para armar la estructura principal y grande de una página.
Referencia: CSS-Tricks - "A Complete Guide to Grid"

Explica display: flex

Flex (Flexbox) es como un collar de perlas. Está diseñado para alinear cosas en una sola línea continua, ya sea en una fila (horizontal) o en una columna (vertical). Se encarga de que todo se estire o se encoja para caber perfectamente en esa única línea.
Referencia: CSS-Tricks - "A Complete Guide to Flexbox"
