1. Introducción a HTML

¿Qué es HTML?

HTML es como el esqueleto de madera de una casa en construcción. No tiene pintura, ni muebles, ni papel tapiz; solo le dice al constructor dónde van las habitaciones, las puertas y las ventanas. Es la estructura cruda de una página web.
Referencia: MDN Web Docs - Introducción a HTML

Explica las etiquetas "head" y "body".

El head (cabeza) es como el cerebro o la configuración oculta de la página. No lo ves en la pantalla, pero guarda información importante como el título que aparece en la pestaña del navegador y los enlaces a los estilos.
El body (cuerpo) es la página web física. Todo lo que ves, lees y a lo que le das clic está dentro del body.
Referencia: W3Schools - HTML Head y MDN HTML Body

Etiquetas Básicas

Explica las etiquetas section, article, aside, header, footer

Imagina que estás leyendo un periódico de papel.
El header es el título gigante hasta arriba con el nombre del periódico.
El footer es la letra chiquita hasta abajo con la fecha y el número de página.
Un section es una página o sección completa, como la "Sección de Deportes".
Un article es una noticia específica e independiente dentro de esa página (una historia que tiene sentido por sí sola).
Un aside es un pequeño recuadro a un lado de la noticia principal con un dato curioso extra.
Referencia: MDN Web Docs - Seccionado de contenido HTML

2. Introducción a CSS

¿Qué es CSS?

Si el HTML es el esqueleto de madera de la casa, el CSS es el decorador de interiores. Es la pintura, las alfombras, las lámparas y los muebles.
Referencia: W3Schools - CSS Introduction

¿Cuál es el propósito de CSS?

Su propósito es hacer que las cosas se vean de una manera específica. Controla los colores, los tipos de letra, los tamaños y en qué parte de la pantalla se acomodan las cosas. (En el caso de esta página, su propósito es lastimar tus ojos).
Referencia: MDN Web Docs - ¿Qué es CSS?

¿Cómo aplicas un CSS a un HTML (explica las 3 opciones)?

  1. En línea (Inline): Es como un tatuaje temporal. Pones el estilo directamente sobre una sola etiqueta usando el atributo style.
  2. Interno (Internal): Es como pintar una sola habitación. Escribes los estilos dentro de la etiqueta <style> en el head, y solo afecta a esa página.
  3. Externo (External): Es como un libro de reglas maestro. Creas un archivo separado (como `estilos.css`) y lo enlazas. ¡Esto le da estilo a cientos de páginas a la vez!
Referencia: MDN Web Docs - Cómo se estructura CSS

¿Qué es un selector? ¿Qué es una regla CSS?

Un selector es "a quién" le estás hablando (Ejemplo: "¡Oigan, todos los párrafos!").
Una regla es "qué" le estás diciendo que haga (Ejemplo: "¡Pónganse de color morado!").
Referencia: MDN Web Docs - Selectores CSS

CSS Básico

Explica los selectores:

  • Selector de Etiqueta (Tag): Apunta a todos los elementos de un mismo tipo. (Ejemplo: "Todas las sillas se vuelven azules").
  • Selector de Clase (Class): Apunta a cualquier cosa que tenga una etiqueta específica. (Ejemplo: "Cualquier persona con el boleto 'VIP' recibe bebida gratis").
  • Selector de Etiqueta y Clase: Muy específico. (Ejemplo: "Solo las SILLAS que tengan el boleto 'VIP' se vuelven azules").
  • Selector de Id: Apunta a UN SOLO elemento único e irrepetible. (Ejemplo: "La persona con el pasaporte 12345, póngase de pie").
Referencia: MDN Web Docs - Selectores de tipo, clase e ID

¿Qué es una clase? ¿Qué es un id? ¿Cómo se relacionan con CSS?

Una clase (class) es como la playera de un equipo de fútbol; muchos elementos diferentes pueden usarla para compartir el mismo estilo. Un id es como el número de placa de un auto; solo UN elemento en toda la página puede tenerlo. CSS usa estos nombres para saber a quién debe aplicarle los colores o tamaños.
Referencia: W3Schools - CSS .class Selector y CSS #id Selector

Explica el Modelo de Caja (Box Model) de CSS. Muestra un ejemplo.

Imagina que cada cosa en una página web es un cuadro de fotografía en la pared.
1. Contenido (Content): La fotografía en sí.
2. Relleno (Padding): El cartón o paspartú blanco que hay entre la foto y el marco.
3. Borde (Border): El marco de madera en sí.
4. Margen (Margin): El espacio de pared vacía que dejas alrededor del marco para que no choque con otros cuadros.

¡Este es el contenido! Nota el padding (espacio dentro del borde), el borde grueso, y el margen (espacio por fuera).
Referencia: MDN Web Docs - El modelo de caja CSS

Explica las media queries

Las media queries son reglas condicionales. Le dicen a la página: "Si la pantalla mide menos de 600 pixeles de ancho (como un celular), pon todo en una columna. Si es más grande (como una laptop), ponlo en dos columnas". Hacen que la página sea responsiva.
Referencia: MDN Web Docs - Consultas de medios (Media queries)

CSS de Diseño (Layout)

Explica display: grid

Grid es como un tablero de ajedrez. Te permite crear estructuras de diseño bidimensionales, acomodando elementos en filas y columnas al mismo tiempo de manera precisa.
Referencia: CSS-Tricks - A Complete Guide to CSS Grid

Explica display: flex

Flex (Flexbox) es unidimensional. Está diseñado para alinear y distribuir el espacio entre los elementos en una sola línea continua, ya sea horizontal (fila) o vertical (columna), permitiendo que se estiren o encojan de forma flexible.
Referencia: CSS-Tricks - A Complete Guide to Flexbox

Juega los 24 niveles de Flexbox Froggy

Flexbox Froggy es un juego educativo donde usas propiedades de CSS Flexbox (como `justify-content` o `align-items`) para ayudar a una rana a llegar a su hoja de nenúfar. Es la mejor herramienta en internet para dominar este sistema de layout.
Referencia: Flexbox Froggy Game - https://flexboxfroggy.com/#es
